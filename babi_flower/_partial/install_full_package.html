Install full package

Using package babi_theme-full-package_1.7.x.x.x.zip

  • Step 1 : Download and use full package you want to install, then extract and upload all content from full package to your server.
  • Step 2 : Create your database in your host under a name YOUR_DATABASE_NAME.
  • Step 3 : Find 1_sample-database.sql and import the sql file to your database you created.
  • Step 4 : Find 2_update-database.sql, edit the last three lines and import the sql file to your database.
    • UPDATE ps_shop_url SET `domain`='YOUR_DOMAIN';
    • UPDATE ps_shop_url SET `domain_ssl`='YOUR_DOMAIN';
    • UPDATE ps_shop_url SET `physical_uri`='/YOUR_BASE_URI/';
    With :
    • YOUR_DOMAIN : is the name of your domain (NOTE : don't contain http:// or https://).
      Example your-domain.com.
    • /YOUR_BASE_URI/ : path linked to sub folder in your site in your server.
      For example, sub folder created to contain your site is new-prestashop, /YOUR_BASE_URI/ is /new-prestashop/.
      On the other hand, if you don't create sub folder to contain your site, you will use : UPDATE ps_shop_url SET `physical_uri`='/';
  • Step 5 : Go to app/config/parameters.php and configure this file with infomation in
    • 'database_host' => 'Database Host'
    • 'database_port' => ''
    • 'database_name' => 'Name Database'
    • 'database_user' => 'Name use'
    • 'database_password' => 'passwords'
    • 'database_prefix' => 'ps_'
  • Step 6 : - Access front-end of your site, if you can't access it, please go to menu Shop Parameters > Traffic & SEO in back-end.
    • The link to access back-end /BASE_URI/psadmin (BASE_URI : the link to front-end your site)
    • Email address: demo@presthemes.com
    • Password: presthemes.
    • You can rename the directory psadmin to change the admin panel’s URL.

      • - Then go to "SET UP URLS", click on "SAVE" to recreate file ".htaccess" automatically.

  • Step 7 : Refesh your front-end, the installation process finishes.
