Babi Theme Configuration

General Setting

1. In your Prestashop Admin Dashboard, navigate to IMPROVE >> Modules >> Modules and Services >> go to Installed modules section, search for “Babi Dashboard ” module then click on Configure.

2. In the Babi Dashboard , choose “General” to configure the parameters in this section. After finishing, you could go to the front-end to view the changes.
